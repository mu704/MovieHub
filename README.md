# MovieHub

## Overview

The Movie Review Web Application is a web-based platform that allows users to discover, review, and rate movies. It provides an interactive and user-friendly environment where movie enthusiasts can explore detailed movie information, share their opinions, and engage with reviews from other users.

## Features

### Browse Movies

Users can browse a collection of movies available on the platform and explore various titles across different genres.

### View Movie Details

Each movie page contains detailed information, including:

* Title
* Genres
* Cast and actors
* Plot summary
* Average user rating

### Read Reviews

Users can read reviews and ratings submitted by other users, providing insights and recommendations for each movie.

### Add Reviews

Registered users can submit their own reviews, including:

* Numerical ratings
* Written comments and feedback

### Search Functionality

The application includes a search feature that allows users to find movies by:

* Title
* Genre
* Actor

### User Profiles

Registered users have personal profiles where they can:

* Manage their reviews
* View favorite movies
* Update account settings

### Rating System

Movies are rated based on user reviews, generating an overall rating that is displayed on each movie's page.

## Technologies Used

### Frontend

* HTML5 for structuring web pages
* CSS3 for styling and responsive design
* JavaScript (ES6+) for client-side functionality, DOM manipulation, and API integration

### Backend and Data Management

* `localStgorage` for managing movie and review data
* JSON for data storage and exchange
